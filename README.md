# カレーマチアプ 🍛

## あなたのカレー性格、診断します。

カレーマチアプは、MBTI式の性格診断をカレーの好みに応用したWebアプリです。25の質問に答えるだけで、あなたのカレー嗜好を5つの軸から分析し、全16タイプのうちどれに当てはまるかを診断します。

## 5つの軸で、カレータイプを診断

- インド派 or 欧風派？
- ドロドロ派 or サラサラ派？
- 辛口派 or 甘口派？
- 具材が主役派 or ルーが主役派？
- こだわり強め派 or なんでもいける派？

## 遊び方

1. 25問の質問に直感で答える（カレーアイコンをタップするだけ）
2. 診断結果であなたのカレータイプが判明
